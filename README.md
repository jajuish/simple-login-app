## Simple login server app

#### Technologies used
- Typescript
- Node.js
- React
- Express
- MongoDB
- JWT

### Run the website

#### Initialize MongoDB connection
1. Set env variable string values inside server/.env

#### Start the frontend server
1. `cd app/`
2. `npm install`
3. `npm start`

#### Start the backend server
1. `cd server/`
2. `npm install`
3. `npm start`

#### Test the application
1. Open your browser
2. Go to http://localhost:8080
